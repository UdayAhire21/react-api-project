This is a simple React application built using Create React App to revise core React concepts.

The app fetches user data from a public API and displays it in a list format. It demonstrates how to work with APIs in React and how to manage different UI states.

## Features
- Fetches data from a public REST API
- Displays loading state while data is being fetched
- Handles and displays error messages
- Renders user list dynamically

## React Concepts Used
- Functional components
- useState for state management
- useEffect for API calls
- Conditional rendering
- List rendering using map

## API Used
- https://jsonplaceholder.typicode.com/users

- React (Create React App)
- JavaScript
